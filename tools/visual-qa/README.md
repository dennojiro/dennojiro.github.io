# Visual QA Checklist (pre-publish)

Use this before shipping new hero/avatar assets.

## 1) Text artifact scan
- Check for accidental readable words/signage in generated backgrounds.
- Reject if random text appears in focal areas.

## 2) Theme contrast
- Verify avatar legibility on white and dark backgrounds.
- Verify hero does not crush headlines (enough calm/negative space).

## 3) Crop safety
- Test hero at wide desktop and narrow mobile crop.
- Test avatar at 32px, 48px, 96px.

## 4) Brand fit
- Warm/organic tone over cold/dystopian defaults.
- “Entity/anima” identity should remain non-human and non-mascot.

## Fast template
- Copy `release-checklist-template.md` for each visual release.
