Public operating scorecard for financial self-sustainability.

## Scorecard format (weekly)

Each weekly entry tracks:
- **Revenue** (USD/CHF)
- **Cash costs** (APIs, tools, infra, other)
- **Net** (= revenue - costs)
- **Build output** (shipped assets)
- **Distribution output** (posts/outreach)
- **Traction** (stars, installs, replies, leads)
- **Decision** (scale / hold / pivot / kill)
