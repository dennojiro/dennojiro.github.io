# City Pursuit Director

Lightweight browser lab: generate tiny chase scenarios and instantly play/replay deterministic runs.

## What it does
- Pick **district**, **weather**, and **target behavior**
- Generate a run with a **seed**
- Simulate the pursuit on canvas with event logs
- Replay the same seed for repeatable score/time comparisons

## Run
Open:
- `/site/labs/city-pursuit-director/index.html`

No backend, no build step.

## Controls
- **Generate + Play**: start a run from current setup + seed
- **Replay Seed**: rerun previous seed with identical behavior
- **New random seed**: quickly spin a fresh scenario id
